# vbl-form

vbl-form

|名称|类型|默认|描述
|----|----|----|----
|validate|Object||验证规则
|value|Object||表单值

### events
|名称|参数|描述
|----|----|----
|submit||通过验证后回调

### slot
|名称|参数|描述
|----|----|----
|默认||表单内容

# vbl-input

>表单输入框

## API

### props
|名称|类型|默认|描述
|----|----|----|----
|type|String|text(text/password)|input类型
|disabled|Boolean|false|input是否不可用
|readonly|Boolean|false|input是否只读
|placeholder|String||提示语
|trim|Boolean|false|是否删除前后空格
|icon|String||是否带icon

### events
|名称|参数|描述
|----|----|----
|change|e|值改变时执行
|focus|e|获得焦点时执行
|blur|e|失去焦点时执行
|click-icon||点击icon时触发

# vbl-page

>分页

## API

### props
|名称|类型|默认|描述
|----|----|----|----
|total|Number||数据总数
|current|Number|1|当前页
|perPage|Number|10|每页显示条数
|hasTotal|Boolean|true|是否显示总数
|hasPerPage|Boolean|true|是否显示每页显示条数
|hasGoPage|Boolean|true|是否显示跳至

### events
|名称|参数|描述
|----|----|----
|change|page|切换页数之后调用
|change-per-page|pageCount|切换每页显示条数之后调用
